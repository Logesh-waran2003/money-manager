List of Features

1. Account Management
   Create an Account: Add a new account (e.g., "HDFC Debit - 2345").

View All Accounts: List all accounts with their balances.

Edit Account: Update account name or type.

Delete Account: Remove an account (only if no transactions are linked).

2. Transaction Management
   Add Transaction: Record a single transaction (income, expense) with amount, category, etc.

Add Transfer: Move money between accounts (e.g., debit to credit).

View Transactions: List transactions with filters (by account, date, category).

Edit Transaction: Modify transaction details.

Delete Transaction: Remove a transaction and adjust balance.

3. Credit Management
   Add Credit: Record money lent or borrowed (amount, beneficiary, dates).

Add Credit Payment: Log a repayment tied to a credit.

View Credits: List all credits with outstanding amounts.

Edit Credit: Update credit details.

Delete Credit: Remove a credit (and its payments).

4. Recurring Spends
   Add Recurring Spend: Define a recurring expense (e.g., Netflix, monthly).

Generate Transactions: Auto-create transactions based on recurring spends.

View Recurring Spends: List all recurring spends with details.

Edit Recurring Spend: Modify frequency, amount, etc.

Delete Recurring Spend: Remove a recurring spend.

5. Insights
   Spending by Category: Show total spending per category (e.g., Food, Travel).

Income vs. Expenses: Compare total income and expenses.

Account Balances: Display current balance for each account.

Outstanding Credits: Show total money lent vs. borrowed (net position).

Recurring Spend Summary: Total cost of recurring spends (monthly/annual).

6. UI/UX Enhancements
   Dashboard: Overview of balances, recent transactions, and insights.

Forms: User-friendly forms for adding/editing data (using Shadcn/UI).

Filters: Filter transactions by date, category, or account.

Responsive Design: Ensure it works on mobile and desktop.

7. Data Integrity
   Balance Updates: Automatically adjust account balances on transaction add/edit/delete.

Transfer Linking: Ensure transfers are paired correctly with transfer_id.

Error Handling: Handle invalid inputs (e.g., negative balance, missing fields).
